# 🤖 AI 产品经理面试复盘

> 把面试录音/笔记扔进来，输出一份结构化的面试复盘报告——四象限分类、逐题打分、改进示范，一站式搞定。

## 它能做什么

- 📂 **四象限自动分类** — 自我介绍与转型动机 / 产品能力与项目落地 / AI 能力与技术判断 / AI 发散认知与成长潜力
- 📊 **逐题评估** — 对照能力矩阵，按 🟢优秀 / 🟡合格 / 🔴需改进 三级打分，每条结论引用原文证据
- ✍️ **改进示范** — 对薄弱题目拆解原始回答 → 重写改进版，标注改进点
- 📄 **结构化输出** — 固定四部分格式（题目罗列 → 逐题评估 → 改进示范 → 反问记录），不跑偏

## 安装

### 方式一：vskills 安装

```bash
npx vskills add DoraRabbitYan/ai-pm-interview-review -y --copy
```

### 方式二：Git 克隆

```bash
git clone https://github.com/DoraRabbitYan/ai-pm-interview-review.git
```

然后放入你的 OpenClaw workspace 的 `skills/` 目录下。

## 快速开始

直接把面试问答记录粘贴过来——不挑格式，能识别出"问题"和"回答"就行。

```
面试官：你为什么从后端转 AI 产品？
我：因为我觉得 AI 是未来趋势……

面试官：介绍一下你主导的最成功的产品项目
我：我主导过广告投放系统的……
```

助手会自动完成分类 → 评估 → 改进 → 输出文档。

## 四象限模型

| 象限 | 考察什么 | 典型问题 |
|------|----------|----------|
| 一 | 自我介绍与转型动机 | "为什么转 AI 产品？""你觉得自己为什么适合？" |
| 二 | 产品能力与项目落地 | "介绍一个你主导的项目""怎么衡量项目成功？" |
| 三 | AI 能力与技术判断 | "LLM 和传统 ML 在产品上有什么区别？""怎么评估模型效果？" |
| 四 | AI 发散认知与成长潜力 | "怎么看 AI 产品未来竞争格局？""最近关注什么新产品？" |

## 输出文档结构

1. **第零部分** — 目标公司与岗位背景
2. **第一部分** — 面试题目按四象限罗列
3. **第二部分** — 逐题评估（按质量降序）
4. **第三部分** — 改进示范（含原始回答拆解 + 改进版）
5. **第四部分** — 候选人反问记录
